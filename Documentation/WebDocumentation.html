PlatformIO commands

  • Init project with specified platform, board... 
    • pio project init --board <boardname> -d <directory path>
  • List all installed boards, platforms, frameworks (could be little slow to wait for answer) 
    • pio boards --json-output
    • pio platform list --json-output
    • pio platform frameworks --json-output
  • List all installed boards using cache files (faster) and linux jq json input processor 
    • cat ~/.platformio/platforms/*/boards/*.json | jq -s '.'
  • Compile and upload current project 
    • pio run -t upload
  • Just compile in debug mode 
    • pio debug
  • Platform - search available 
    • pio platform search arduino
    • pio platform search stm*
  • Platform - install new one 
    • pio platform install atmelavr
